# Firebeetle2C6_Hotwater
This project is to measure the Water entering a Wetback on a fire and Water exiting. There is a third temperature probe to measure the Hot water tank temperature.

A Project using the Arduino IDE to 

3 x DS18B20 Temperature sensors to measure fire place Wetback

As this project uses a C6 it needed https://forum.arduino.cc/t/onewire-library-esp32/1266351/4
